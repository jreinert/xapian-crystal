# xapian-crystal

Xapian bindings for crystal

## Installation

1. install [xapian](http://xapian.org/)
2. Compile and install https://github.com/endlessm/xapian-glib
3. Add xapian to `Projectfile`

```crystal
deps do
  github "jreinert/xapian-crystal"
end
```
